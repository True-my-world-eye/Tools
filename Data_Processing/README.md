**项目简介**
- 面向高校或企业数据筛选的工具集，支持对 Excel/CSV 中“专业”等字段进行匹配筛选，同时支持多列、多条件的综合判定
- 提供两种形态：桌面图形应用（GUI）与命令行工具（CLI），满足不同数据规模与使用场景

**核心能力**
- 多条件筛选：以条件文件（CSV）描述规则，支持文本、数值、枚举、正则、编码与模糊匹配，AND/OR/加权组合
- 专业列筛选（旧版）：在仅有“专业要求文本”的场景下快速筛选，保留编码优先与相似度策略
- 批量处理：多文件输入，逐文件输出与合并输出，支持追加与去重
- 进度与日志：运行时展示进度条与关键日志，帮助观察整体处理情况与结果命中

**两种使用方式**
- GUI（目录：`gui/`）
  - 适合一般数据规模与可视化操作
  - 提供文件选择、条件导入/编辑、模式切换（多条件/旧版）、输出设置与运行控制
  - 支持 Sheet 多表合并、逐文件与合并输出、去重键与追加模式、可选审计列
- CLI（目录：`cli/`）
  - 适合百万级行数据与自动化批处理
  - 参数集中在代码顶部配置，终端输出进度与日志
  - 支持分块处理、Sheet 多表合并、向量化条件评估与合并写出

**目录结构**
- `gui/`：图形应用与说明文档
- `cli/`：命令行脚本与说明文档
- `requirements.txt`：依赖声明（含可选加速库）
- `require.txt`、`resume_require.txt`、`combined_conditions_full.csv`：示例数据或条件文件
- `TMT_FIGUREINFO*.xlsx`：示例源数据与处理结果

**依赖与平台**
- Python 3.9+；Windows 与 macOS 通用
- 依赖通过 `requirements.txt` 安装

**何时选择 GUI 或 CLI**
- GUI：需要直观操作、条件少于数百条、数据规模中等、偏向一次性处理或交付可执行程序
- CLI：需要脚本化或自动化、大规模数据（百万行级）、追求稳定的批处理与较少的交互

**典型流程**
- 准备条件文件（CSV）或旧版专业要求文本
- 选择处理模式（多条件或旧版）
- 添加数据文件与设置输出选项（逐文件/合并、去重、追加）
- 启动处理，查看进度与日志；结果写出到指定目录

**备注**
- GUI 与 CLI 的核心筛选思路一致；GUI提供更直观的交互，CLI强调高性能与可扩展
- 如需深度定制（字段映射、条件模板、日志保存到文件等），可在对应目录的说明文档中参考扩展建议
